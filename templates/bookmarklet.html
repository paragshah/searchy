+ Add to Searchy

How to install

  1. Make sure your browser's bookmarks bar is visible.
  2. Drag the blue "+ Add to Searchy" button above to your bookmarks bar.
  3. Visit any page and click the bookmarklet to save it to Searchy.

Notes

  • Searchy must be running on localhost:{{ port }} for the bookmarklet to work.
  • The bookmarklet opens a small popup window where you can add tags before saving.
